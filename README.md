#  PowerBI_Dashboard-Project

# 🚨 Crime Data Analysis of India | Power BI Dashboard 🇮🇳

## 📌 Project Overview
This project focuses on analyzing crime data across India using **Power BI**.  
The goal is to transform raw data into meaningful insights through interactive dashboards and visualizations.

---

## 🎯 Objectives
- Analyze crime distribution across different states
- Identify major crime categories and their contribution
- Compare crime rates per lakh population
- Build an interactive and user-friendly dashboard

---

## 🔍 Project Highlights
- Performed **data cleaning and preprocessing**
- Built **data models and relationships** in Power BI
- Used **DAX functions** for calculated measures
- Designed interactive visuals:
  - Map visualization (state-wise crime distribution)
  - Bar charts (category comparison)
  - Pie charts (proportion analysis)
  - KPI cards (summary metrics)

---

## 📊 Key Insights
- Crime distribution varies significantly across states
- **Offences against Human Body** and **IPC crimes** contribute the highest number of cases
- Certain states show consistently higher crime rates
- Crime rate per lakh population provides better comparison than total cases

---

## 🛠️ Technologies Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation  
- Data Visualization  

---

## 📁 Dataset
- Source: Government/Open dataset (Crime records of India)
- Contains data such as:
  - State
  - Crime Category
  - Total Cases
  - Crime Rate per Lakh
  - Year

2. Open it in **Power BI Desktop**
